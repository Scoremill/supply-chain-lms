# Supply Chain LMS — Supply Chain Academy

## Tech Stack
- Next.js 14 (App Router) + Tailwind CSS
- Prisma + PostgreSQL (Neon)
- NextAuth v4
- Radix UI, TanStack Query, Chart.js
- PDF-lib for certificate generation
- Source on GitHub (Scoremill/supply-chain-lms)

## What This Is
A learning management system for construction supply chain training. Cloned from HowToBuildAHome (construction-lms) with supply chain curriculum. Course platform with assessments, progress tracking, and certificate generation.
